# XSS-Scanner
Simple XSS scanner.

Usage: `python3 xss_scanner.py -u URL [flags] [arguments]` 

`-t` to choose type  
`-p` to set specific payloads

It is possible to scan multiple URLs from a file.

`python3 xss_scanner.py -l FILENAME`
